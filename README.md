# Caesar Cipher Tool

## 📝 Description
A Python-based implementation of the classic Caesar Cipher. This tool is designed for CTF (Capture The Flag) competitions to quickly encrypt or decrypt messages found in "Cryptography" challenges.

## ✨ Features
- **Bi-directional:** Handles both encryption and decryption.
- **Case Sensitivity:** Preserves the casing of the original message.
- **Symbol Protection:** Ignores spaces, numbers, and special characters.

## 🚀 How to Run
1. Ensure you have Python 3.x installed.
2. Run the script:
   ```bash
   python caesar.py
